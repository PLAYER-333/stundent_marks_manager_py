Student Marks Manager (Python) 🎓
A lightweight, console-based application designed to manage student records, calculate grades, and ensure secure data access. This project was developed as part of my beginner Python journey to practice data structures, control flow, and deployment.

🚀 Features
Add Student Records: Quickly store student names and marks in a dynamic dictionary.

Secure Data Access: The "View Student" and "Check Result" modules are password-protected to simulate basic administrative security.

Automated Grading Logic: Instantly calculates results (PASS/FAIL) and assigns grades (A+ to D) based on the marks entered.

Memory Efficient: Refactored from recursion to iterative loops to ensure stability and better memory management.

Standalone Executable: Includes a .exe version for Windows users who don't have Python installed.

🛠️ Tech Stack
Language: Python 3.x

Data Storage: Python Dictionaries (In-memory)

Deployment: PyInstaller (for .exe conversion)
